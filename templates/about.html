About Me

This is what I do.

This is a blog platform built using Python and Flask. This blog allows CRUD operations. You can create new posts, edit existing ones, delete, and read all posts.

The database used for storing data is postgresql. The ORM is SQLAlchemy.

The front end uses bootstrap and wtforms with some Python logic mixed in through the Jinja template engine language.

You can also register users. Only registered users can comment on the posts. For authentication I have used werkzeug security for hashing and salting of passwords.
